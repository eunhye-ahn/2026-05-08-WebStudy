# HTML & CSS 정리

---

##  HTML 태그 정리

### 1. 텍스트 구조화 태그

#### 블록과 단락

- `<h1>` ~ `<h6>` : 제목 출력
- `<p>` : 단락 나누기
- `<br>` : 줄바꿈

#### 의미 & 스타일

- `<strong>` / `<b>` : 굵게
- `<del>` / `<ins>` : 취소선 / 밑줄
- `<sup>` / `<sub>` : 위 첨자 / 아래 첨자
- `<pre>` : 입력 포맷 유지

#### 특수문자

- `&nbsp;` : 공백
- `&lt;` : `<`
- `&gt;` : `>`

---

### 2. 네비게이션과 하이퍼링크 `<a>`

- `<a href="#news">뉴스</a>` : 문서 내 위치로 스크롤 즉시 이동 (긴 페이지 탐색에 유용)
- `target` 속성
  - `"_self"` : 현재 브라우저 탭에서 화면 전환
  - `"_blank"` : 새로운 탭을 열어 출력
- `href="URL"` : 이동할 대상 파일명이나 외부 웹 주소 지정

---

### 3. 정보 구조화: 목록 태그

- `<ul>` : 순서가 없는 목록 (주로 네비게이션에 활용) → `<li>`
- `<ol>` : 순서가 있는 목록 → `<li>`
- `<dl>` : 제목-설명이 매칭된 데이터 목록 → `<dt>` → `<dd>`

---

### 4. 테이블

```html

   ... 
   ... 
   ... 
```

| 태그 | 설명 |
|------|------|
| `<tr>` | 가로 한 줄 |
| `<th>` | 제목 셀 (가운데 정렬, 굵게) |
| `<td>` | 일반 데이터 셀 (기본 왼쪽 정렬) |

- `colspan="2"` : 가로 방향(열) 셀 병합
- `rowspan="6"` : 세로 방향(행) 셀 병합

## CSS 정리

### 블록 vs 인라인

- `<div>`, `<p>` : 부모 요소의 전체 가로 공간을 100% 차지 → `width`, `height`로 크기 자유롭게 조절 가능
- `<span>` : 딱 자신의 콘텐츠 크기만큼 영역 차지 → 인라인 배치, `width`와 `height` 적용 X
  - `display: inline-block` → 나란히 배치되면서 블록 크기도 마음대로 조절 가능

---

### 선택자

| 선택자 | 설명 |
|--------|------|
| `A B` | 하위 선택자 |
| `A > B` | 자식 선택자 |
| `A + B` | 인접 형제 |
| `A ~ B` | 일반 형제 |

---

### 속성 선택자

| 선택자 | 설명 |
|--------|------|
| `[속성=값]` | 정확히 일치 |
| `[속성~=값]` | 단어가 포함됨 |
| `[속성\|=값]` | 값이 정확히 일치하거나 하이픈으로 시작 |
| `[속성^=값]` | 해당 값으로 시작하는 요소 |
| `[속성$=값]` | 해당 값으로 끝나는 요소 |
| `[속성*=값]` | 어디든 포함된 요소 |

### 상태 및 구조 가상 클래스

#### 링크 상태 순서
:link → :visited → :hover → :active

#### 구조 선택자

- `:first-child`
- `:nth-child`
- `:last-child`

#### 가상 요소 (DOM에 없는 콘텐츠 생성)

- `::before` : 요소 앞
- `::after` : 요소 뒤
